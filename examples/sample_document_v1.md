# Sample Document

This is a sample document to demonstrate the Document Evolution System.

## Introduction

The Document Evolution System helps you track changes to your documents over time.

## Features

- Version tracking
- Change detection
- History management

## Usage

Track this document:
```bash
python3 cli.py track examples/sample_document.md -m "Initial version"
```

Update after changes:
```bash
python3 cli.py update examples/sample_document.md -m "Added content"
```

View history:
```bash
python3 cli.py history examples/sample_document.md
```
